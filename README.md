# Сервис коротких ссылок

Сервис для создания коротких ссылок с веб-интерфейсом и REST API. Позволяет сокращать длинные URL, создавать кастомные коды и отслеживать статистику переходов.

## Возможности

- **Веб-интерфейс** - удобная форма для создания коротких ссылок
- **REST API** - программное создание и управление ссылками
- **Кастомные коды** - возможность задать свой вариант короткой ссылки
- **Статистика** - отслеживание количества переходов по ссылкам
- **Автоматический редирект** - переход по короткой ссылке на оригинальный URL
- **Защита** - Rate limiting, CORS, валидация URL и кастомных кодов

## Технологии

- **FastAPI** - современный веб-фреймворк
- **PostgreSQL** - база данных (создается автоматически при запуске приложения)
- **SQLAlchemy** - ORM для работы с БД
- **Docker** - контейнеризация приложения
- **Jinja2** - шаблонизатор для веб-интерфейса
- **Pydantic** - валидация данных
- **SlowAPI** - rate limiting
- **Pytest** - тестирование

## Требования

- Python 3.11+
- PostgreSQL 17+
- Docker

## Инструкция по установке и запуску сервиса

- Клонирование репозитория:
`git clone https://github.com/SibAlive/URL-shortener.git`
- Вход в папку проекта
`cd URL-shortener`
- Настройка переменных окружения (файл .env необходимо заполнить своими данными):
`cp .env.example .env`
- Запуск docker контейнера
`docker compose up -d --build`

## Инструкция по эксплуатации сервиса
### Сервис состоит из двух частей - Web и API

## API: 
- Создание короткой ссылки
### POST http://localhost:8000/shorten
Запрос:
```
{
    "url": "https://example.com/example-url",
    "custom_code": "your_short_code"  // опционально
}
```

Ответ:
```
{
    "short_id": "your_short_code" // или сгенерированный код
}
```

- Редирект по короткой ссылке
### GET http://localhost:8000/{short_id}

Ответ
```
{
    "original_url": "https://example.com/example-url"
}
```

- Получение статистики
### GET http://localhost:8000/stats/{short_id}

Ответ
```
{
    "clicks": 3
}
```

## Web интерфейс:
- Веб интерфейс доступен по пути http://localhost:8000/web/
- Веб интерфейс имеет те же возможности, что и API
